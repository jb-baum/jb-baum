## Relational Landscapes

Relational Landscapes is a geospatial analytical framework for diagnosing ecological structure and landscape dynamics using multispectral satellite observations.

Rather than treating landscapes as static baselines, the framework interprets them as relational ecological systems whose internal structure evolves through time.

The system analyzes relationships among vegetation productivity, moisture expression, terrain structure, and disturbance signals to characterize ecological state and identify structural divergence across heterogeneous landscapes.

Applications are implemented using Google Earth Engine and cloud geospatial analysis pipelines.

Live platform  
https://relational.land

## RLX Covariance Kinematics

RLX is a monitoring system for analyzing ecological regime dynamics in satellite time-series data.

The method models ecological state as a covariance structure among environmental signals and tracks how that structure moves through time using rolling-window analysis.

Diagnostics include:

• feature-space displacement  
• regime velocity  
• regime acceleration  
• covariance volatility  
• dominant signal drivers  
• observation support governance  

The goal is to detect structural ecological instability prior to large disturbance events such as wildfire.
